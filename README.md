# OllamaChatApp
My first Android app using Ollama! 🚀
Built with love in Termux.

## Features
- Uses Ollama for local AI chat
- Simple Android UI (built in Termux)
- Includes pre-packaged app.zip

## How to Run
1. Clone this repo
2. Open in Android Studio or Termux
3. Run `./gradlew installDebug`
